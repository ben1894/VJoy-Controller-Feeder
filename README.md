# VJoy Arduino Controller Feeder
Receives Bluetooth data from an Arduino and gets processed as an HID device using the VJoy SDK for Dolphin Emulator (or any application that uses HID game controls).

# Setup
Arduino Code is included in the BluetoothFeeder folder. For use with the HC-05 bluetooth module. Wiring description in code.

vJoy must be installed to run.

vJoyInterface.dll must be included with application (installer coming).
